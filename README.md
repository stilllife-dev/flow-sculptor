# FLOW SCULPTOR
### Generative Art for Interference and Filtration

## Demo
[ここに公開後のURLを貼る]

**FLOW SCULPTOR** は、繊細な複数のラインが干渉し合うことで、建築的かつ有機的な「層（レイヤー）」を描き出すジェネレーティブ・アート・ツールです。

## Concept: Filtration (濾過)

## Features
- **Generative Interaction**: 5つのスライダーで密度の変化や干渉をコントロール。
- **Live Animation**: 蠢く（うごめく）ような有機的なアニメーション。
- **SVG Export**: 生成されたアートをベクター形式で即座に保存可能。

## How to use
1. スライダーを動かして、ラインの干渉を調整します。
2. ANIMATEトグルをONにして、流れの変化を観察します。
3. 気に入った形ができたら、DOWNLOAD SVGをクリックして保存します。
